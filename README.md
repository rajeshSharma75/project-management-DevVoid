## üöÄ Installation & Setup

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/project-management-DevVoid.git
cd project-management-DevVoid
```

### Step 2: Backend Setup

#### Install Dependencies
```bash
cd server
npm install
```

#### Create Environment File
Create a `.env` file in the `server` directory:

```env
PORT=5000
NODE_ENV=development

# MongoDB Atlas Connection
MONGODB_URI=mongodb+srv://<username>:<password>@cluster0.xxxxx.mongodb.net/project-management?retryWrites=true&w=majority

# JWT Configuration (CHANGE THESE IN PRODUCTION!)
JWT_SECRET=your-super-secret-jwt-key-change-this-in-production
JWT_EXPIRE=1h
JWT_REFRESH_SECRET=your-super-secret-refresh-key-change-this-in-production
JWT_REFRESH_EXPIRE=7d

# Google Gemini AI API Key
GEMINI_API_KEY=your-gemini-api-key-here

# CORS Configuration
CLIENT_URL=http://localhost:3000
```

**Important Notes:**
- Replace `<username>` and `<password>` with your MongoDB Atlas credentials
- Replace `xxxxx` with your MongoDB cluster ID
- Get your Gemini API key from [Google AI Studio](https://aistudio.google.com/app/apikey)
- The application uses **Gemini 2.5 Flash** model (`gemini-2.5-flash`)

#### Start the Backend Server
```bash
npm run dev
```

‚úÖ Server will start on **http://localhost:5000**

### Step 3: Frontend Setup

#### Install Dependencies
```bash
cd ../client
npm install
```

#### Create Environment File
Create a `.env` file in the `client` directory:

```env
REACT_APP_API_URL=http://localhost:5000/api
```

#### Start the React App
```bash
npm start
```

‚úÖ Frontend will open on **http://localhost:3000**

## üìñ Usage Guide

### 1Ô∏è‚É£ Register/Login

- **Drag and drop** tasks between columns:
  - üìù **To Do** - New tasks
  - üîÑ **In Progress** - Work in progress
  - ‚úÖ **Done** - Completed tasks
- Task status updates automatically when moved
- Visual indicators for overdue tasks and priorities

## üîß Troubleshooting

### MongoDB Connection Error
**Error:** `MongoServerError: bad auth`
- ‚úÖ Check username and password in `MONGODB_URI`
- ‚úÖ Ensure IP address is whitelisted in MongoDB Atlas (use `0.0.0.0/0` for all IPs)
- ‚úÖ Verify database user has read/write permissions

### CORS Errors
**Error:** `Access-Control-Allow-Origin`
- ‚úÖ Verify `CLIENT_URL` in server `.env` matches your frontend URL
- ‚úÖ Ensure both frontend and backend servers are running
- ‚úÖ Check CORS configuration in `server.js`

### Gemini AI Not Working
**Error:** `[404 Not Found] models/... is not found`
- ‚úÖ Verify your `GEMINI_API_KEY` is correct
- ‚úÖ Ensure model name is `gemini-2.5-flash`
- ‚úÖ Check API quota in [Google AI Studio](https://aistudio.google.com/)
- ‚úÖ Verify internet connection

### Port Already in Use
**Error:** `EADDRINUSE: address already in use`

**Windows:**
```bash
netstat -ano | findstr :5000
taskkill /PID <PID> /F
```

**Linux/Mac:**
```bash
lsof -ti:5000 | xargs kill -9
```

### JWT Token Errors
**Error:** `JsonWebTokenError: invalid token`
- ‚úÖ Clear browser localStorage and login again
- ‚úÖ Verify `JWT_SECRET` and `JWT_REFRESH_SECRET` are set
- ‚úÖ Check token expiry settings
